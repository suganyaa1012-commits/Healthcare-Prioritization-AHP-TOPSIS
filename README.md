
# Healthcare Resource Prioritization Model Using AHP & TOPSIS

## Project Overview
This project helps policymakers identify which Indian states should be prioritized for healthcare resource allocation.

### Methods Used:
- AHP for weighting
- TOPSIS for ranking
- Streamlit dashboard

## Criteria Used
- Hospital Beds
- Life Expectancy
- Death Rate
- Disease Burden
- Population Density
- Poverty Score

## Tools Used
Python, Pandas, NumPy, Streamlit, Plotly

## Files
- app.py
- requirements.txt
- Healthcare_TOPSIS_Ranked_Fixed.xlsx

## Run App

streamlit run app.py
